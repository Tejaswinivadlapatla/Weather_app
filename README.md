# Weather App
##  About the Project
- A modern and responsive Weather Application built using HTML, CSS, and JavaScript. This application fetches real-time weather data using the OpenWeather API and displays detailed weather information for any city.

- The Weather App is a responsive web application developed using HTML, CSS, and JavaScript that provides real-time weather information for any city worldwide. It integrates the OpenWeather API to fetch live weather data and presents it through a modern, user-friendly interface.

- The application allows users to search for a city and instantly view important weather details such as temperature, weather condition, humidity, wind speed, atmospheric pressure, visibility, and the current date and time. It also features an attractive world map background and a glassmorphism-based user interface to enhance the user experience.

- This project demonstrates the use of API integration, asynchronous JavaScript (fetch, async/await), DOM manipulation, responsive web design, and error handling for invalid city searches. It is suitable as a beginner-to-intermediate front-end web development project and showcases practical skills in building real-world applications.

## Features
### 
- Search weather by city name
- Real-time temperature
- Weather condition
- Humidity
- Wind speed
- Atmospheric pressure
- Visibility
- Current date and time
- Attractive world map background
- Responsive and user-friendly interface
- Error handling for invalid city names
  
## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
